# Create + Immersive Engineering progression (suggested)

Use **quests** (FTB Quests, etc.) or **advancements** to enforce order; these are **recipe ideas** only.

## Chapter 1 — Foundations
- Vanilla tools → first ores.
- **Immersive Engineering:** Coke Oven → Blast Furnace → **Steel**.
- **Create:** Andesite Alloy → **Hand Crank** / **Water Wheel** → basic machines.

## Chapter 2 — Power
- IE: **Kinetic Dynamo** + water/wind, then **LV/MV** wiring.
- Create: route **SU** to pressing / mixing you need for IE parts (optional cross-crafting).

## Chapter 3 — Automation
- Create: **Brass**, **Precision Mechanism** (sequenced assembly), belts.
- IE: **Assembler**, **Metal Press**, **Arc Furnace** (as your pack allows).

## Chapter 4 — Advanced industry
- IE: **Heavy Engineering** blocks, **Steel plates**, **Electron Tubes** (if using IE electronics).
- Create: **Electron Tube** (Create), **Sturdy Sheets** / late components (pack-dependent).

## Chapter 5 — Endgame (Changed tech)
- Gate **Stasis Chamber**, **Exoskeleton**, **Exoskeleton Charger** behind the scripts in `modpack/kubejs` or `modpack/scripts` (Create + IE + nether stars + Furmutage blocks).

Adjust item IDs in the scripts to match **your exact mod versions**.
