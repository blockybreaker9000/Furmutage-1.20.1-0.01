# Sound Files

Place your sound files in this directory with the following names:

## Required Sound Files:

1. **night_ambient_1.ogg** - First night ambient sound
2. **night_ambient_2.ogg** - Second night ambient sound  
3. **night_ambient_3.ogg** - Third night ambient sound
4. **morning_chime.ogg** - Morning chime sound

## Sound File Requirements:

- Format: `.ogg` (OGG Vorbis)
- Recommended sample rate: 44100 Hz or 22050 Hz
- Recommended bitrate: 128 kbps or higher
- Mono or stereo is fine

## When Sounds Play:

- **Night Ambient Sounds**: Play randomly during night time (6:30 PM to 5:30 AM) approximately every 10 minutes
- **Morning Chime**: Plays once per day at dawn (around 6:00 AM)

## Notes:

- The sounds will automatically be loaded by the mod once placed in this directory
- Make sure the file names match exactly (case-sensitive)
- You can use any audio editing software to convert your sounds to OGG format
